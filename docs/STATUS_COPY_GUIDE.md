# Status Copy Guide

Short, consistent status text for UI messages and toasts.

## Success
- `X created`
- `X updated`
- `X deleted`
- `X saved`
- `X sent`
- `X applied`
- `X exported`
- `X imported`
- `X uploaded`
- `X downloaded`
- `X copied`

## Failure
- `X create failed`
- `X update failed`
- `X delete failed`
- `X save failed`
- `X send failed`
- `X apply failed`
- `X export failed`
- `X import failed`
- `X upload failed`
- `X download failed`
- `X copy failed`
- `X load failed`

## Availability
- `X not available`
- `X unavailable`

## Auth/Permission
- `Authentication required`
- `Permission denied`
